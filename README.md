# AI Lost & Found System

An AI-powered campus lost and found management system that helps students recover lost items using NLP similarity matching.

## Features

- Report Lost Items
- Report Found Items
- AI Text Similarity Matching (NLP)
- Student Dashboard
- Admin Dashboard
- Image Upload Support
- Match Similarity Percentage

## Tech Stack

Frontend
- HTML
- CSS
- JavaScript

Backend
- PHP
- MySQL

AI / NLP
- Python
- Scikit-learn
- TF-IDF Similarity
